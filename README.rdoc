== README

1) TheGraft – designed a simple user-interfaced app that allows even the most inept internet user to quickly find the major contributors to a politicians’ campaign.  Designed ultimately to be mobile app to be used in booth, but is scalable to be incorporated into a systemic resource.
	  Resources: Angular.js written in coffescript front-end interfaced with a Rails backend using JSONP to connect to the Opensecret 	  API, styling in Bootstrap, and D3 charting, provides SPA usability -

	   http://thegraft.herokuapp.com/
